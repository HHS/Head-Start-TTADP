# Proposed Schema for Collaboration Reports MVP

## Status

Proposed

## Context

[TTAHUB-3473|https://jira.acf.gov/browse/TTAHUB-3473] is tracking the work to create an MVP (Minium Viable Products) for Collaboration (collab) Reports. Collab reports are a new entity within the TTA Hub and as such new database constructs ( tables, indices etc.) are needed. This ADR will document a proposed schema to serve as basis for the database design, prior to implementation.
